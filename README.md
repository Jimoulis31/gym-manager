# Gym Manager

A desktop gym tracking app built with Python and Tkinter.

## Features
- Log workouts and exercises (sets, reps, kg)
- Track progress over time with charts
- Personal records auto-detected
- Workout calendar view
- Workout templates
- Body weight tracker
- Rest timer
- Export all data to CSV

## Requirements
- Python 3.8+
- matplotlib — `pip install matplotlib`
- ttkthemes — `pip install ttkthemes`

## How to Run
python gym_manager.py
