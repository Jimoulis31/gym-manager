# Gym Manager

A desktop workout tracking application built with Python and Tkinter, 
designed to help you log, track, and analyse your gym progress over time.

![Gym Manager Screenshot](images/screenshot.png)

## Features

- **Workout Logging** — Create workouts and add exercises with sets, reps and weight
- **Progress Tracking** — Visualise your kg progression over time with charts
- **Personal Records** — Automatically detects your all-time best lift per exercise
- **Workout Calendar** — See which days you trained at a glance
- **Workout Templates** — Save a workout layout and reuse it instantly
- **Body Weight Tracker** — Log your body weight and see it charted over time
- **Rest Timer** — Countdown timer with presets for rest between sets
- **CSV Export** — Export all your workout data to a spreadsheet

## Requirements

- Python 3.8+
- matplotlib &nbsp;→&nbsp; `pip install matplotlib`
- ttkthemes &nbsp;→&nbsp; `pip install ttkthemes`

## How to Run

1. Clone the repository:
```
   git clone https://github.com/Jimoulis31/gym-manager.git
```
2. Install dependencies:
```
   pip install matplotlib ttkthemes
```
3. Run the app:
```
   python GymTrackingApp.py
```
